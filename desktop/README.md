# 🎮 FitGirl Browser - Desktop App

A beautiful, fast, and keyboard-friendly desktop application for browsing FitGirl Repacks. Inspired by PSVita's PKGj interface.

## ✨ Features

- **PKGj-Inspired UI** - Nostalgic Vita-style interface
- **Full Keyboard Navigation** - Use your keyboard for everything
- **Skinnable Themes** - Built-in themes + custom theme support
- **Fast Search** - Instant filtering as you type
- **Torrent Integration** - Built-in torrent client (coming soon)
- **Cross-Platform** - Windows, macOS, Linux

## 🚀 Getting Started

### Prerequisites

- Node.js (v18+)
- Rust (latest stable)
- The `repacks.db` database (generated by the crawler)

### Installation

```bash
# Install dependencies
npm install

# Copy database from root to desktop folder
# (On Windows PowerShell)
Copy-Item ..\repacks.db .\repacks.db

# (On Linux/Mac)
cp ../repacks.db ./repacks.db

# Run in development mode
npm run tauri dev

# Build for production
npm run tauri build
```

## ⌨️ Keyboard Shortcuts

### Navigation

- `↑` / `↓` or `J` / `K` - Navigate game list
- `←` / `→` or `H` / `L` - Switch sidebar/list focus
- `Home` / `End` - Jump to first/last game
- `PgUp` / `PgDn` - Page through list

### Actions

- `/` or `Ctrl+F` - Focus search box
- `Enter` - Open first magnet link
- `1-9` - Open specific magnet link by number
- `C` - Copy first magnet link to clipboard
- `R` or `F5` - Refresh database

### Pages

- `Ctrl+1` - Browse page
- `Ctrl+2` - Downloads page
- `Ctrl+3` - Settings page
- `Ctrl+4` - Stats page

### Application

- `T` - Quick theme switcher
- `F11` - Fullscreen
- `Q` or `Ctrl+Q` - Quit

## 🎨 Themes

### Built-in Themes

- **Nord** - Cool, arctic-inspired theme (default)
- **Dracula** - Dark theme with purple accents

### Creating Custom Themes

Themes are defined in JSON format with CSS color variables:

```json
{
  "name": "My Theme",
  "author": "Your Name",
  "version": "1.0.0",
  "colors": {
    "background": "#1a1a2e",
    "primary": "#e94560",
    "text": "#eaeaea",
    ...
  }
}
```

Place custom themes in `themes/custom/` and they'll appear in the theme selector.

## 🔧 Development

### Frontend (Svelte)

The UI is built with SvelteKit and TailwindCSS. All components are in `src/lib/components/`.

### Backend (Rust)

The backend uses Tauri 2.0 and provides commands for:

- Database queries (search, get details)
- Magnet link handling
- Clipboard operations
- File system access

### Adding New Commands

1. Define the command in `src-tauri/src/commands.rs`
2. Add it to the invoke handler in `src-tauri/src/lib.rs`
3. Call it from Svelte using `invoke('command_name', { args })`

## 📦 Building

### Development Build

```bash
npm run tauri dev
```

### Production Build

```bash
npm run tauri build
```

The built application will be in `src-tauri/target/release/`.

## 🐛 Troubleshooting

### Database not found

Copy `repacks.db` from the project root into the `desktop/` folder:

```bash
# From the desktop folder
Copy-Item ..\repacks.db .\repacks.db  # Windows
cp ../repacks.db ./repacks.db          # Linux/Mac
```

### Themes not loading

Check that theme JSON files are valid and in the `themes/` directory.

### Keyboard shortcuts not working

Make sure no input element has focus. Press `Esc` to clear focus.
