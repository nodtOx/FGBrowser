### Built-in Themes

- **Nord** - Cool, arctic-inspired theme (default)
- **Dracula** - Dark theme with purple accents

### Creating Custom Themes

Themes are defined in JSON format with CSS color variables:

```json
{
  "name": "My Theme",
  "author": "Your Name",
  "version": "1.0.0",
  "colors": {
    "background": "#1a1a2e",
    "primary": "#e94560",
    "text": "#eaeaea",
    ...
  }
}
```

Place custom themes in `themes/custom/` and they'll appear in the theme selector.

## ğŸ”§ Development

### Frontend (Svelte)

The UI is built with SvelteKit and TailwindCSS. All components are in `src/lib/components/`.

### Backend (Rust)

The backend uses Tauri 2.0 and provides commands for:

- Database queries (search, get details)
- Magnet link handling
- Clipboard operations
- File system access

### Adding New Commands

1. Define the command in `src-tauri/src/commands.rs`
2. Add it to the invoke handler in `src-tauri/src/lib.rs`
3. Call it from Svelte using `invoke('command_name', { args })`

## ğŸ› Troubleshooting

### Database not found

Copy `repacks.db` from the project root into the `desktop/` folder:

```bash
# From the desktop folder
Copy-Item ..\repacks.db .\repacks.db  # Windows
cp ../repacks.db ./repacks.db          # Linux/Mac
```

### Themes not loading

Check that theme JSON files are valid and in the `themes/` directory.

### Keyboard shortcuts not working

Make sure no input element has focus. Press `Esc` to clear focus.
